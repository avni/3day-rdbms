VM Tips:

  • Shift-click on the XTerm launcher button to get a second terminal.
  • Use the X version of emacs for copy/paste. Select text and Ctrl-C to copy, Edit -> Paste in emacs to paste.

Android Studio Tips:

  • Unlock the virtual phone by clicking on the lock and swiping up.
  • The VM webserver is reachable from the emulator at http://10.0.2.2:8080/.
  • Using the emulator:
    • Hotkeys: Back=Esc, Home=Home (fn-left arrow on mac), swiping: click and drag across the emulator surface.
    • Keyboard entry: click on the native Android touch keyboard (the painful way) or enable keyboard support in AVD Manager (Tools -> Android -> AVD Manager -> (choose device) -> Edit -> Advanced Settings -> Keyboard Input).
